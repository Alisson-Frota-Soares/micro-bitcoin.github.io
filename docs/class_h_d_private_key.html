HDPrivateKey Class Reference

HD Private Key class. Derived from PrivateKey class. Works according to bip32, bip39 and slip32. You can generate the key from mnemonic or seed, derive children and hardened children. xprv and xpub methods return strings according to slip32, xprv/xpub for bip44, yprv/ypub for bip49 and zprv/zpub for bip84.
